## 🚀 Запуск

### Локально

```powershell
cd D:\qwen\language-practice\frontend
npm run dev
# → http://127.0.0.1:5173
```

### Card Sync Server (на сервере)

```bash
ssh -p 2222 root@192.168.199.222
cd /var/www/english-master/server
npm install
npm start &
```

### Деплой на сервер

```powershell
cd D:\qwen\language-practice
.\deploy.ps1
```

```powershell
cd D:\qwen\language-practice
git add -A; git commit -m "описание"; git push
```
